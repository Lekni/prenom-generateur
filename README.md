# Générateur de Prénoms

Ce projet est un générateur de prénoms en Python utilisant **Streamlit**. L'application permet de changer des syllabes de prénoms et de générer de nouveaux prénoms en fonction d'une liste fournie par l'utilisateur.

## Fonctionnalités

- Entrez un prénom et choisissez une partie du prénom à changer (initiale, médiane, finale).
- Collez une liste de prénoms pour générer de nouveaux prénoms.
- Affiche jusqu'à 20 prénoms générés.
